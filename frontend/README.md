# ForgeTrack Frontend

This directory contains the React + Vite frontend for ForgeTrack.

For the full project documentation, including features, tech stack, and roadmap, please see the [main README.md](../README.md).

## 🚀 Development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

## 🛠 Tech Stack
- React 18
- Vite
- Tailwind CSS
- Framer Motion
- Lucide Icons
